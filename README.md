# Webpack fileinfo-loader

Webpack fileinfo-loader with:

- Webpack 5
- Typescript
- loader
- fs.stats

### Installation

```bash
yarn add -D fileinfo-loader
```

### Start developing and serve your app

```bash
npm start
```

### Build your application

```bash
npm run build
```

### Run unit tests

```bash
npm run test
```

### Run coverage

```bash
npm run coverage
```

### Docker

Or simply run the example using docker:

```bash
docker-compose up
```
